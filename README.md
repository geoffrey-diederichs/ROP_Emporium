# ROP Emporium

Write-ups of the [ROP Emporium](https://ropemporium.com) challenges teaching about how to exploit buffer overflows.
All of those challenges have been solved on a `x86_64` system.

## [ret2win](./01_ret2win)

- Executing a function.
- Realigning the stack.

## [split](./02_split)

- Modifying arguments to a function call.

## [callme](./03_callme)

- Calling a function by it's PLT entry.

## [write4](./04_write4)

- Writing over memory.
